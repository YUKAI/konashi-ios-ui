## Requirements

iOS 13.0+

## Installation

### CocoaPods

```ruby
pod 'konashi-ios-ui'
```

### Swift Package Manager

```swift
dependencies: [
    .package(name: "KonashiUI", url: "https://github.com/YUKAI/konashi-ios-ui.git", from: "1.0.0"),
    /// ...
]
```
